Creating procedural materials and a small sci-fi diorama using Substance Designer

Procedural Art

4 sci-fi materials made with Substance Designer

Project Details

Team

Solo

Duration

6 Weeks

Software

Substance 3D Designer, Unity HDRP

Engine

Unity 6 (HDRP)

Project Overview

For this project, I learned how to use Substance Designer by creating four separate procedural materials based on a sci-fi theme. These materials were implemented in a Unity HDRP scene, each with at least five exposed parameters to allow real-time customization.

As part of the assignment, I also created several high-resolution beauty renders to showcase the materials in detail.

Development

The development process started with gathering references and creating rough prototypes in Substance Designer. I began by blocking out the main shapes using the height map, then moved smaller details into the normal map to optimize performance.

Ambient occlusion was generated from the height and normal data to add depth. Wear and tear were first created in grayscale before applying color, allowing better control over intensity.

Each material includes at least five exposed parameters, mostly affecting shape and color, making them highly adjustable in real time.

Afterward, the materials were imported into Unity and placed in a circular floating diorama. I used HDRP lighting, post-processing, and Polybrush to blend materials naturally.

One of the main challenges was balancing visual quality with flexibility, as well as ensuring all materials worked well together through lighting and composition.

What I Learned

  • How to create procedural materials using Substance 3D Designer
  • How to use exposed parameters for real-time customization
  • Improved workflow in Unity HDRP (lighting and post-processing)
  • How to blend materials using Polybrush
  • Better understanding of non-destructive workflows
