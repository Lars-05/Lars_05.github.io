Game Overview

Project Growth is a desktop application where you collect and grow adorable buddies using currency earned by completing tasks within a set time limit. Its goal is to reduce procrastination and motivate users to finish their work more efficiently.

Development

Project Growth is designed to be supportive without becoming a distraction. The application uses soft pastel colors to avoid overstimulation and maintain a calm visual experience. Sound effects were avoided as much as possible to prevent disrupting the user’s focus.

One of the biggest technical challenges was enabling the buddies to move naturally across a 2D surface. This was solved by making a plane and shaping it into the walkable area using ProBuilder, generating a NavMesh for pathfinding on it.

What I Learned

  • Learned how to develop a desktop application using unity
