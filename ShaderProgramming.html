Project Overview


For this project individual project I scripted multiple shaders using HLSL and ShaderGraph, these shaders range from visually pleasing ones (Disolve, Hologram, Intersection Shaders) to very technical ones (Custom normal calculated lighting shaders).

For this project, I also created a tree with its mesh and UVs generated through an algorithm which can take multiple parameters

What Did I learn?


• Learned how to manipulate 2D textures and color data in C#, HLSL and ShaderGraph to create different visual effects.
• Practiced creating and controlling rotations in both 2D and 3D projects.
• Used dot and cross products for tasks like basic lighting, calculating mesh normals, UV mapping, and programming movement or AI behavior in games.
• Gained a better understanding of how 3D meshes work, including geometry, normals, and UVs.
• Learned to write and experiment with vertex and fragment shaders using Unity URP, HLSL and ShaderGraph.
• Improved my understanding of coordinate spaces and how to transform between them with matrices and vector bases.
• Applied matrices to perform affine transformations such as moving, scaling, and rotating objects.
