Project Overview


In this project, I scripted multiple shaders using HLSL and ShaderGraph. These ranged from more pretty shaders such as dissolve, hologram, and intersection shaders, to more technical shaders like custom lighting calculations with live normal calculation. I also created a procedural tree generator, which builds the tree meshes and UVs algorithmically based on several parameters.

What Did I learn?


• Manipulation of 2D textures and color data using C#, HLSL, and ShaderGraph.

• Applying dot and cross products for tasks like basic lighting, calculating mesh normals, UV mapping, and controlling movement or AI behavior in games.

• Understanding 3D meshes, including geometry, normals, and UV generation.

• Scripting vertex and fragment shaders with Unity URP, HLSL, and ShaderGraph.

• Knowledge of coordinate spaces and transforming between them using matrices and vector bases.

• Using matrices to perform transformations such as translation, scaling, and rotation of objects.
